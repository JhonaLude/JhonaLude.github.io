Descripción del proyecto

Friendly es un innovador brazo robótico diseñado para recoger basura, este brazo es controlado mediante una aplicación desarrollada en AppInventor, que se comunica con Friendly a través de Bluetooth. La aplicación permite a los usuarios mover el brazo robótico, abrir y cerrar la pinza para recoger basura.

El proyecto consiste en la creación de un brazo robótico que pueda ser controlado por una aplicación móvil, por medio de conexión Wi-Fi y Bluetooth para su control.
